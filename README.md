# Enhanced Expense Tracker

The Enhanced Expense Tracker is a Java Swing-based desktop application that helps users record and manage daily expenses. It allows users to add expense details such as category, amount, and date, display them in a table, and calculate total spending in real time with budget tracking using Object-Oriented Programming principles.

## Features
- Add and categorize expenses
- View expenses in table format
- Budget tracking
- Real-time total calculation

## Technologies Used
- Java
- Java Swing
- OOP
